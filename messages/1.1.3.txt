====================
     Changelog
====================

Thanks for downloading this package!

Changes from v1.1.2 to v1.1.3 include:

- Support for the C++11 syntax package
- A new snippet, try catch block. Triggered by "try"
- Added a menu to view documentation under Preferences > Package Settings

As always, if you would like to add a snippet feel free to send me a pull
request on GitHub: https://github.com/Rapptz/cpp-sublime-snippet
